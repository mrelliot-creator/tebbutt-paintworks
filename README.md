# Tebbutt & Son Paintwork’s Website

Fully free, static website for our painting & decorating business.

## How to update the site
- Add new photos → Upload them to the `assets/gallery` folder (they appear automatically)
- Change text or contact details → Edit the .html files directly in GitHub
- The estimate form sends emails to YOUR_EMAIL_HERE via FormSubmit

Live site: https://mrelliot-creator.github.io/tebbutt-paintworks/
