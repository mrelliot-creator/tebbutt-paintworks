PLACE ALL YOUR PROJECT PHOTOS HERE

How to add new images (no code changes needed):

1. Go to your GitHub repository in a browser.
2. Navigate to the folder: assets/gallery
3. Click "Add file" → "Upload files"
4. Drag and drop any JPG, JPEG, PNG or WEBP images of your painting work.
5. Write a short commit message and click "Commit changes".

The live website will automatically show the new photos within seconds – no editing required!

Recommended filenames (for nice captions):
- kitchen-before.jpg
- living-room-after.webp
- exterior-house.jpg
- feature-wall.jpg

You can upload as many photos as you want. The gallery updates itself!
